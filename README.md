# Лабораторна робота: Дослідження колекцій в Python

Цей проект є виконанням лабораторної роботи з дослідження класів-колекцій, узагальнених типів та ітераторів з використанням мови Python.

## Структура проекту

- `structures.py`: Містить визначення класів `ManagedString`, `Node` та `BinaryTree`.
- `main.py`: Головний скрипт, який демонструє роботу всіх завдань.

## Як запустити

1.  Переконайтесь, що у вас встановлено Python 3.
2.  Помістіть файли `structures.py` та `main.py` в одну папку.
3.  Відкрийте термінал або командний рядок у цій папці.
4.  Виконайте команду:

    ```bash
    python main.py
    ```

5.  У консолі з'явиться результат виконання всіх завдань лабораторної роботи.
